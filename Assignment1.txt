Purpose:
The purpose of this assignment is to build the front-end of an Ice Cream Shop Ordering system.
The focus will be on designing and organizing HTML components to simulate an Ice Cream
Shop Ordering system.

•

The submission should be a single compressed file (.zip) containing the complete source
code and resources (e.g., images), ready for execution without modifications.

•

Name the submitted zip file as: Assignment1_ENSF381_Section#_Group#. For
example: Assignment1_ENSF381_L01_Group01.

•

Include the group members’ names and UCIDs at the top of the main HTML file
(index.html) as source code comments.

Students are expected to use only the HTML tags and properties introduced during
lectures and mentioned in this assignment ONLY.

The goal is to create a visually appealing and user-friendly Ice Cream Shop Ordering System
with the following pages: Homepage, Login Page, Menu View, Customize Order, and Order
Summary. Focus solely on the HTML structure and CSS styling to create pages that are
professional, visually appealing, and user-friendly.
NOTE: All styles must be within the styles.css file.
